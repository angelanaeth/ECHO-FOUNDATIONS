Zones placeholder · canonical

Heart-rate zones are the foundational anchor for everything we do. The values below are pulled directly from the ECHO Manual and will always match — if Angela changes a zone in ECHO, Foundations updates automatically.

Canonical reference (live from ECHO): these values are auto-synced. Do not edit by hand.
  • Z1 Recovery / Easy — <81% of LTHR
  • Z2 Aerobic Endurance — 81-89% of LTHR
  • Z3 Tempo — 90-93% of LTHR
  • Z4 Lactate Threshold — 94-99% of LTHR
  • Z5 VO2 / Anaerobic — 100%+ of LTHR

Block Types placeholder · canonical

Foundational block-type vocabulary. The full ECHO Manual layers in advanced variants — these intro blocks are the starting point.

Canonical reference (live from ECHO):
  • Base — Build aerobic capacity and durability. Mostly Z1-Z2 volume.
  • Build — Introduce threshold and tempo work on top of aerobic base. Z3-Z4 emphasis.
  • Peak — Race-specific intensity and durability. Mix of Z3-Z5 with race-pace work.
  • Taper — Reduce volume while preserving sharpness ahead of A-race.
  • Recovery — Active recovery week, ~50-65% normal volume, Z1-Z2 only.
